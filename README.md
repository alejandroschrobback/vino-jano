# Vino Jano · CAVA Anita y Jano

Dashboard de inventario para una cava INELRO 177 BOT STEEL DOOR.
Single-file React app (Babel standalone, persistencia en localStorage, sync vía Gist privado).

## Uso

Abrí `cava-anita-jano-v5.html` en el navegador. No hace falta servidor.

## Sync entre dispositivos

Configurá en cada dispositivo el mismo Personal Access Token de GitHub (scope `gist`) y el mismo Gist ID. La app baja en bootstrap y mantiene auto-sync.
